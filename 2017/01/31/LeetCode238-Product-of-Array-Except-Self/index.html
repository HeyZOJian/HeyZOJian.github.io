LeetCode238. Product of Array Except Self

题意

给一个数组A，生成一个数组B，B[i] = A[0] x A[1] x..x A[i-1] x A[i+1] x..x A[n]
时间复杂度O(N)，不能使用除法

思路

仔细看一下，其实每次计算B[i]就是前面i-1个元素相乘，乘上后面i+1个元素相乘。
所以只要预处理前缀积和后缀积即可。
